Nome do projeto: My Education CODED / CODED

Alunos integrantes: Kauã Pablo Souza de Lima / Cauê Silva do Nascimento / Davi Fernandes Domingues Soares

Descrição do sistema simulado: O sistema simula a parte de login de usuario (no contexto do projeto seria os alunos e professores), acesso do admin (que tem a permissão de CRUD; Criar, Ver, Atualizar
e Deletar usuarios do sistema, representando a propria intituição de ensino responsavel por gerenciar seus alunos e professores).

Link de deploy: 
- Link do modelo antigo (apenas HTML, CSS e JAVASCRIPT): https://coded-aluno.netlify.app/ obs: para login: usuario = aluno / senha = 1234.

- Link do modelo novo ( já com php e certas mudanças de layout): https://codeddev.free.nf/coded_prototipo_ofc/app/view/ obs: o novo modelo possui dois logins diferentes; o de aluno (caso esteja
cadastrado no banco de dados) e o de admin: usuario = admin / senha = 0000.
